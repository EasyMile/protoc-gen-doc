CentOS 7 repository for protoc-gen-doc

This is a x86_64-only CentOS 7 repository for protoc-gen-doc.

Enable the repository and install the package with:

curl -o /etc/yum.repos.d/protoc-gen-doc.repo https://estan.github.io/protoc-gen-doc/protoc-gen-doc.repo
yum install -y epel-release
yum install -y protoc-gen-doc

The EPEL repository is needed to get recent versions of qt5-qtbase and protobuf-devel.
